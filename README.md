# Capstone Project

This end-to-end data analysis project predicts music genres across a 50,000-record Spotify dataset using label-specific data imputation, feature extraction, and fine-tuned classification models including neural networks, random forests, and gradient boosting.

## Live Demo
https://github.com/xq675/capstone

## Technologies
- Python
- Scikit-Learn
- Pandas
- Numpy
- Matplotlib

## Key Features
- Engineered an end-to-end multi-class classification pipeline to predict music genres across a 50,000-record Spotify dataset using fine-tuned Multi-Layer Perceptron, Random Forest, and Gradient Boosting
- Implemented domain-specific data imputation by grouping features by target class and replacing corrupted/missing values with localized medians to preserve data integrity and eliminate class imbalance
- Executed dimensionality reduction using Principal Component Analysis to mitigate severe feature multicollinearity
